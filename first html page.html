HTML stands for HyperText Markup Language and serves as the standard language used to create and structure web pages. It acts as the skeleton or structural blueprint of almost every website on the Internet, determining how text, images, and links are organized before they are displayed by a web browser.

Core Concepts of HTML

1. HyperText: Refers to the hyperlinks that connect web pages to one another.

2. Markup Language: A system for annotating a document with tags that tell a browser how to format and display the text.

3. Not a Programming Language: It does not handle logic, calculations, or functionality; it only structures content.

4. The Web Trifecta: HTML builds the structure. Developers pair it with CSS for presentation/styling and JavaScript for interactivity.

How HTML Works (Tags & Elements)

HTML uses tags enclosed in angle brackets (< and >) to identify different pieces of content. Most elements consist of an opening tag, the content, and a closing tag (which includes a forward slash).

Common HTML Tags

1. <html> : The root element that wraps all the content on the page.

2. <head> : Contains invisible meta-information about the page, like the title.

3. <body> : Contains all the visible content, such as text, images, and hyperlinks.

4. <h1> to <h6> : Used to create headings.

5. <p> : Used to define a paragraph.

6. <a> : Used to create hyperlinks.

7. <img> : Used to display images.

Basic Structure Example

<!DOCTYPE html>
<html>
<head>
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is a short paragraph explaining the page content.</p>
</body>
</html>
